# ペルソナ定義

AIキャリア診断サービス「5問でわかる、あなたのキャリア」のLPおよびStage1設計の前提として、キャリアに悩む代表的な3タイプを定義する。

---

## ペルソナA：就活中の学生（意思決定の軸が固まっていない）

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 佐藤 みなみ（22歳・大学4年） |
| **状況** | 就活は進めているが、業界・職種が多すぎて比較軸が曖昧。自己分析シートは書いたが「自分らしい軸」に自信がない。 |
| **ゴール** | 面接で説明できるキャリアのストーリーと、第一志望群の根拠を短時間で整理したい。 |
| **課題** | 情報過多で疲弊。長文の自己分析ツールは続かない。親や友人の意見に流されやすい。 |
| **デジタル行動** | SNS・求人サイト・大学のキャリアセンターを行き来。スマホでの閲覧が中心。 |
| **LPで期待すること** | 「5問」「AI」で負担が小さいと感じ、試しやすい。結果が「ロードマップ」として見えると納得しやすい。 |
| **懸念** | 無料診断の質、個人情報、就活に悪影響がないか。 |

---

## ペルソナB：転職を検討している社会人（現職との比較が必要）

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 田中 健太（29歳・事務職・入社5年目） |
| **状況** | 現職に不満はあるが、転職先のイメージが漠然としている。スキルギャップと年収・ワークライフバランスを同時に考えて頭が止まりがち。 |
| **ゴール** | 転職エージェントに会う前に、自分の方向性を言語化し、論点を整理したい。 |
| **課題** | 時間がない。長い適性検査やワークショップは現実的でない。家族・住宅ローンなど制約がある。 |
| **デジタル行動** | 通勤中に記事・求人をチェック。PCとスマホの両方で利用。 |
| **LPで期待すること** | 短時間で「次の一手」が見えるイメージ。社会的信用（説明文の具体性・透明性）があると安心。 |
| **懸念** | 診断が浅く、転職サイトと同じ一般論に終わらないか。 |

---

## ペルソナC：キャリアに漠然と不安がある20〜30代（行動は迷いがち）

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 鈴木 あかり（26歳・非正規／フリーランス／異業種） |
| **状況** | 「このままでいいのか」は感じているが、就活でも転職でもない。学び直し・副業・資格など選択肢が多く、優先順位がつけられない。 |
| **ゴール** | 自分の価値観・強みに沿った「次の3〜6ヶ月」の行動イメージが欲しい。 |
| **課題** | 正解がない領域の悩みで、検索しても答えが出ない。一人で考えるとループする。 |
| **デジタル行動** | 夜間・休日にスマホで調べる。体験型・診断系コンテンツに反応しやすい。 |
| **LPで期待すること** | 重くないトーンで「まず試せる」導線。結果が「地図」のように見えると続きを見たくなる。 |
| **懸念** | 精神論や占いっぽさ。継続課金や強い営業につながる印象。 |

---

## 3ペルソナ共通の設計示唆（LP観点）

- **短時間・低摩擦**をヒーローと特徴で一貫して伝える（5問の根拠を一言で補足できるとよい）。
- **成果物のイメージ**（ロードマップ／次の一手）をビジュアルまたはコピーで具体化する。
- **安心材料**（プライバシー、無料範囲、AIの役割）をフッター付近または特徴の1枠で触れると転換率が上がりやすい。
