# LP動作確認チェックリスト（Stage1）

対象：AIキャリア診断サービス LP（Stage1）

## 前提

- Stage1のCTAは **ページ遷移しない**（クリック時に **Snackbarで Coming Soon（Stage2以降で提供予定）** を表示）

## 確認環境

- **ブラウザ**: Chrome / Edge
- **表示**: PC / タブレット / スマホ（レスポンシブ）

## チェックリスト

### 1. 各セクションの表示確認

- **Heroセクション**
  - キャッチコピー「5問でわかる、あなたのキャリア」が表示される
  - サブコピー「AIがあなたに最適なキャリアロードマップを提案します」が表示される
  - CTAボタンが表示される
  - 右側（または下部）にビジュアル（カード風）が表示される
- **Featuresセクション（3カード）**
  - 3つの特徴カードが表示される（タイトル/説明/アイコン）
  - アイコンが欠けずに表示される
- **Stepsセクション（3ステップ）**
  - Step 1〜3が順に表示される（Step 1: 回答 / Step 2: AI分析 / Step 3: 結果表示）
  - レイアウト崩れ（重なり・はみ出し）がない
- **Bottom CTAセクション**
  - 見出しが表示される
  - CTAボタンが表示される
- **Footer**
  - 著作権表示が表示される
  - （実装されている場合）Stage1スコープ注記が表示される

### 2. レスポンシブ表示（PC/タブレット/スマホ）

- **PC（md以上想定）**
  - Heroが2カラム（左テキスト/右ビジュアル）で表示される
  - 文字サイズ・余白が過度に詰まらない
- **タブレット（sm想定）**
  - カード/ステップの折り返しが自然で、要素が欠けない
  - CTAが画面内で押しやすいサイズになっている
- **スマホ（xs想定）**
  - Heroが1カラム縦積みになり、CTAが押しやすい位置・サイズになっている
  - テキストの改行が不自然にならない（1文字だけの孤立など）
  - 横スクロールが発生しない

### 3. CTAボタンの動作

- **Hero CTA**
  - クリックしてもページ遷移しない
  - Snackbar（Coming Soon）が表示される
  - 連打しても表示が破綻しない（多重表示/レイアウト崩れがない）
- **Bottom CTA**
  - クリックしてもページ遷移しない
  - Snackbar（Coming Soon）が表示される
  - 連打しても表示が破綻しない

### 4. リンク遷移

- **LP内リンク（存在する場合）**
  - クリックで正しい位置へスクロール/遷移する
  - タブ移動やフォーカス移動（キーボード操作）で到達できる
- **外部リンク（存在する場合）**
  - 正しいURLへ遷移する
  - 期待どおりの遷移方式（同一タブ/別タブ）になっている

### 5. 画像・アイコンの表示

- 画像・アイコンが 404 にならずに表示される
- 高解像度端末でもぼやけが目立たない（必要に応じて）
- ダーク背景/明るい背景で視認性が落ちない（配色の前提がある場合）
